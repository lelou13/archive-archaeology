Archaeology Database Entries

1.    The correct format for the entry of catalog numbers is as follows:  "00000.00".  In other words, the required format consists of five digits to the left of the decimal point, and two digits to the right.  For example, the catalog number "00345.00" would be the equivalent of catalog number "345".

2.    In the event that you have entered a catalog number that produces an error message, or you find a duplication of the catalog number in the database, simply use the two digits to the right of the decimal point to assign a new "temporary" number.  For example, if catalog number 00345.00 has already been used in the database, but the catalog card itself bears the same number, simply renumber the catalog number "00345.01" or "00345.02"...taking care to use the two digits to the right of the decimal point as your new temporary catalog number.

3.    Correct format for all date entries is as follows: June 19, 1999 would be entered as "6-19-99" and June 19, 2001 would be entered as "6-19-01".

4.    If you enter a record and select "New Record" and a pop-up window indicates a "New Record Error." You may have problems with any of the following: a. Catalog number has already been taken by another record. In that event, use the last two digits (after the decimal point) to change the number by one numeral (e.g. from 00989.00 to 00989.01). If this doesn't work, and you receive another "New Record Error," then you can change the last two digits one or two other times until the record enters (e.g. 00989.02 or 00989.03). If you change the catalog number in the database record, you must also change the new catalog number on the artifact bag tag as well.

b. Date field is either incomplete (e.g. 6-98) or incorrect (e.g. 6-00-98). Use correct format (e.g. "6-15-98") or leave field blank.

c. Catalog number is entered incorrectly or in the wrong format. You should have five number spaces to the left of the decimal point, and only two number spaces to the right of the decimal point.

d. Before changing the number due to an error, please be sure that all fields or entry areas are entered completely and in correct format (e.g. 6-15-98 for dates; 00983.00 for catalog numbers). 5.    Anytime that you change the catalog number in the artifact database, you must also change the number on the artifact bag tags.

6.    If you encounter a "Artifact Type" or "Material Type" that is not already included in the pop-up list in the database, please make a note of that fact and Dr. Mendoza will add the artifact or material type to the database for use in future record entries.

7.    If you have a suggestion, or have noticed some error, or limitation, with the database, please e-mail Dr. Mendoza by clicking on the "ASTV E-mail" button on the top-right portion of the Institute of Archaeology Web Site at http://archaeology.monterey.edu

8.    When adding catalog numbers to a catalog card, please use either one of the numerical stamps provided by Professor Mendoza. Please change the number stamp by one digit each time that you assign a number to a bag tag or catalog card. Do not exceed the 99 number in any series of 100. In other words, if the number stamp that you are using is set at 658 at the time you begin using it, do not use a number higher than 699. In addition, when you are finished using the stamp for the day, please note on a card kept with the stamp the last number that you entered that day (please include both last number, date, and time of last use).

9.    When providing catalog numbers, or preparing descriptions for the database, please be prepared to remove the artifact(s) from the artifact bag so that you can more closely examine the artifact on each side for the purposes of providing a more complete description. Always return the artifact to the artifact bag in the same location that you found it. The proper placement of the artifact should be to the back or behind the written portions of the catalog card. The decorative or more distinctive side of the artifact should placed so as to face outward to the back of the artifact bag. Ultimately, all catalog information should be visible on one side of the bag, and on the other side of the bag, the more distinctive features of the artifact contained therein should be visible to the viewer without the need to remove the artifact from the artifact bag.

10.    At the end of your database session, please shut down Netscape or other Internet browser so that the password protected portions of the site will not be tampered with by the next user.
