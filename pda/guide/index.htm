Users Guide for Access Point Selection



Click here for iPaq User's Guide, or click here for the Toshiba User's Guide.


Compaq iPaq e3950

From the Today screen click Start > Settings
This should bring you to the Control Panel. Click the "System" tab.



Click the "Wireless Network" icon.

At this point the "Wireless Networking" dialog should pop up.
A dropdown menu will display two Wireless profiles, "ASTV" and "CSUMB".



If you will be at or near the CSUMB campus,
and within range of the wireless network, choose the "CSUMB" profile.

If you will be using the iPaq away from the campus, and within range
of a remote (wireless) base station, select the "ASTV" profile.






Toshiba e740

From the Today screen click Start > Settings
This should bring you to the Control Panel. Click the "System" tab.



Click the "Wireless LAN Utility" icon.

At this point the "Wireless Lan Utility" dialog should popup.
There should be two dropdown menus, Mode and SSID.



Make sure Mode is set to "Infrastructure".

If you will be using the Toshiba at or near the CSUMB campus and within
range of the wireless network, select "CSUMB" from the SSID dropdown menu.

If you will be using the Toshiba away from the campus, and within range of a
remote (wireless) base station, choose "ASTV2" from the SSID dropdown menu.
