Database Codes

Materials and links contained here provide guidelines terminology for artifact and specimen codes used in the Wireless and or Online Database of the Institute for Archaeology. When entering data online, or converting raw pHp MySQL server tabled data for those specimens entered, it may be useful to first review artifact and specimen codes so as to anticipate categories and types to be entered or available. When conducting analysis of the raw data from the php MySQL database, please bear in mind that the Artifact and Material Type lists have been converted to those numbers noted in the following tables. In order to convert numerical designations into Artifact and Material Type names and terms, simply select the Excel data column for either Artifact or Material Type and convert the numerical designations back to text.
